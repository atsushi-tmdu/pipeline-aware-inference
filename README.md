# Pipeline-aware inference after clinical machine-learning model search

This repository implements pipeline-aware max-statistic calibration for
statistical inference after adaptive clinical machine-learning model search.

The repository contains code for:

- simulation studies of type I error, power, and selection-induced optimism;
- analyses of candidate-model dependence and effective search size;
- the SUPPORT2 clinical-data application;
- reproduction of the manuscript figures and tables.

## Repository status

The accompanying manuscript is in preparation. Code and documentation are
being organized for reproducibility before submission.

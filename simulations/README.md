# Simulation studies

This directory contains the simulation programs used to evaluate
pipeline-level global-null inference after adaptive model search.

## Organization

- `phase1/`: selection-induced optimism under the global null
- `phase2/`: signal strength, event count, feature selection, and metric analyses
- `phase3/`: primary independent-null evaluation of type I error and power
- `phase3b/`: candidate-library dependence and effective search size

The final analyses used independent null-reference and evaluation random-number
streams. Detailed settings are described in the accompanying manuscript and
supplementary material.
